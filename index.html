Projet S103 - Installation et Configuration d'un SGBD sur Raspberry Pi

Dans le cadre du module S103 en première année de BUT Informatique, j'ai travaillé en équipe pour installer et configurer un système de gestion de base de données (SGBD) sur un Raspberry Pi 400. Ce projet pratique nous a permis d'acquérir des compétences essentielles en administration système et base de données.

Objectifs principaux :

  • Installation d'un système d'exploitation (Raspberry Pi OS) sur une carte SD vierge
  • Configuration réseau du Raspberry Pi via une connexion Ethernet
  • Installation et configuration d'un SGBD (MariaDB)
  • Création d'une base de données "CAMPING" avec ses tables et données
  • Automatisation du démarrage des services

Compétences acquises :

  • Administration système Linux (création d'utilisateurs, gestion des permissions)
  • Gestion de paquets avec apt et configuration de services
  • Manipulation de bases de données SQL (création de tables, insertion de données)
  • Résolution de problèmes matériels et logiciels
  • Rédaction de documentation technique en anglais

Le projet a été évalué via un script automatique vérifiant l'installation complète, un QCM individuel et un rapport technique de 5 pages en anglais. Cette expérience m'a permis de comprendre l'importance de la documentation et du travail d'équipe dans les projets informatiques.
